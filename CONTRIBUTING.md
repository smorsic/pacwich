# Contributing

Thank you for your interest in contributing to **`pacwich`**.

Contributions are welcome. Well-suited contributions include:

- Bug fixes
- Documentation corrections or clarifications
- Small, focused improvements

Please ensure all CI checks pass before requesting a review. Pull requests should automatically run the required checks.

The project is actively developed with a clear [roadmap](https://pacwich.dev/roadmap).

If you have larger ideas than the suggested contributions above, consider heading to our [discussions](https://github.com/smorsic/pacwich/discussions) to see if others have posted similar ideas or to get feedback first.

## Getting Started

Run `bun install` at the repo root. You can run `mise install` afterwards to install dev tool versions (Bun, node, pnpm, etc.).

After install you should have an AGENTS.md file that is used for AI agent guidance and can serve as quick docs for yourself as well.

Some local commands you may run first to help with checks:

- `bun format` - run Prettier
- `bun type-check` - check the project's type safety
- `bun lint` - lint the project
- `bun pw test` - run tests for the pacwich package
- `bun pw build` - run the build test.
