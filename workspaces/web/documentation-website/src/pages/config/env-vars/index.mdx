# Configuration: Environment Variables

## Settable Environment Variables

Set these environment variables to override default values used. The [project configuration](/config/project) will take precedence over environment variables, 
and any values set explicitly in CLI or API arguments will take precedence over both.

<ul className="env-vars-list">
{Object.values(ENV_VARS_METADATA).map(envVar => <div key={envVar.envVarName}>
  <li className="env-var-item" key={envVar.envVarName}>
    <code>{envVar.envVarName}</code>
    <div>{envVar.description}</div>
    {envVar.projectConfigDefaultsKey && <div>If <code>{envVar.projectConfigDefaultsKey}</code> is set in the <Link href="/config/project" className="inline-link">project configuration</Link>, it will take precedence.</div>}
  </li>
</div>)}
</ul>
