You can suppress warnings via:
* The `--suppress-warnings` [CLI global option](/cli/global-options#suppress-warnings),
* The `suppressWarnings` [project config](/config/project#defaults) default,
* The `suppressWarnings` [option in the TS API](/api/reference#createfilesystemproject),
* The [environment variable](/config/env-vars) <code>{getUserEnvVarName('suppressWarningsDefault')}</code>.

These are additive with each other. Using the API's `setSuppressWarnings` [function](/api/reference#setsuppresswarnings) will override the suppressions globally.
