# AI Integrations: MCP Server

`pacwich` provides its own MCP server via `npx pacwich mcp-server`. This can be used with any tooling that supports MCP.

When you set up the MCP server with your tool, an agent has access to the server's **resources** that provide it complete documentation,
the same documentation also available via [AGENTS.md files](/ai/agents) or [skills](/ai/skills).

Agents that read the server instructions on initialization will automatically have knowledge of `pacwich`
as a package with a brief overview and introduction to the CLI, kept very brief due to tools' propensity to
truncate these instructions.
