# AI Integrations

`bun-workspaces` provides several AI integrations to help you manage your Bun monorepo.

## AGENTS.md

`bun-workspaces` provides an **AGENTS.md** file that documents the package for AI agents,
which can be integrated into CLAUDE.md or other AI agent configuration files for immediate 
context.

See the [page on AGENTS.md](/ai/agents) for more information.
