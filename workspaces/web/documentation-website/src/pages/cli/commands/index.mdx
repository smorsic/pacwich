#### JSON Output

Using the `--json-outfile=path/to/results.json` option will output the results of all script runs to a JSON file.
The file will be written relative to the project root, so it is affected by the [`--cwd` global option](/cli/global-options#working-directory).
This is the same data as would be returned by the equivalent API, [`FileSystemProject.runScriptAcrossWorkspaces()`](/api/reference#runscriptacrossworkspaces).
