# Changelog

## 0.0.1

- pacwich Workspaces tree view (Explorer sidebar): browse workspaces, expand for path/alias/tags/scripts, reveal in Explorer, open `package.json` or `pacwich.workspace.*` config.
- `pacwich: Go to Workspace` quick-pick command.
- Auto-refreshing on relevant file changes (package.json, lockfiles, pacwich config files), no manual refresh required.
- `pacwich.projectRoot` setting for projects nested in a subdirectory of the opened folder.
