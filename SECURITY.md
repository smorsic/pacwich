# Security

## Practices

There is an [official page](https://pacwich.dev/security) describing the security practices used for this package, with links to audit source code where possible.

## Privately Report A Vulnerability

Use the official [GitHub vulnerability report for the repository](https://github.com/smorsic/pacwich/security/advisories/new).

`pacwich`'s own `doctor` command (`pacwich doctor` if you have a global install, `npx pacwich doctor`, or `bunx pacwich doctor`) can help produce diagnostic information to
supply in your report.
